Server Management Application (Blazor Server - .NET 8)
This project is a Blazor Server application built using .NET 8. It provides an interactive web-based dashboard to manage and monitor server operations, including checking server health (CPU, memory, and disk usage) and starting/stopping services in real-time.

Features
Real-time Server Monitoring: Monitor CPU usage, memory, and other server health metrics.
Service Management: Start and stop server services directly from the control panel.
Real-time Notifications: Get live updates on server status using SignalR.
Interactive Dashboard: Built with Blazor components to provide a dynamic user interface.
Technologies Used
.NET 8 with Blazor Server
SignalR for real-time communication
C# for backend server logic
HTML & CSS for UI components
